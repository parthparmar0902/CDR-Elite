# Parth Parmar — Trading Skills

Custom AI agent skills for institutional-grade trading research on XAUUSD, BTC, ETH, and SOL.

## Skills

| Skill | Description |
|-------|-------------|
| [cdr-elite](skills/cdr-elite/) | Caveman + DeepResearch + QuantResearch composite skill. Institutional research, compressed output. |

## Installation

```bash
npx skills add parth-parmar/cdr-elite
```

Or install a specific skill:

```bash
npx skills add parth-parmar/cdr-elite --skill cdr-elite
```

## Usage

Trigger CDR Elite with:

```
/cdr
cdr mode
elite research
deep quant research
```
